# Digital Business Card 📇

This is a clean and professional Digital Business Card website built as part of my frontend development learning journey.

## 🚀 Features
- Fully responsive layout centered using **CSS Flexbox**.
- Clean and modern UI with custom hover animations on social links.
- Structured with semantic HTML5 elements.

## 🛠️ Tech Stack
- **HTML5** (Structure)
- **CSS3** (Flexbox, Hover Effects, Typography)

## 🎯 What I Learned from this Project
- How to horizontally and vertically center elements using Flexbox.
- Debugging layout overflows (fixing horizontal row vs vertical column bugs).
- Creating smooth transitions on interactive button states.
